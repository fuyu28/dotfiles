## i3 全体の設定内容まとめ

### 基本設定・見た目

* **Modキー**

  * `$mod = Mod4`（Windowsキー / Superキー）

* **共通変数**

  * `$term = wezterm`（標準ターミナル）
  * `$shots = $HOME/Pictures/Screenshots`（※今は設定内では未使用）

* **ウィンドウ枠**

  * デフォルトのタイルウィンドウ枠は **pixel 1 → 最後に pixel 2 に上書き**
  * フローティングウィンドウの枠は `normal`
  * `hide_edge_borders none` で画面端の枠も隠さない
  * `focus_follows_mouse no` でマウスオーバーではフォーカスを移さない

* **フォント**

  * ウィンドウタイトル・バー用に
    `Iosevka Nerd Font, Noto Sans CJK JP 11` を指定

* **フローティングの移動**

  * `$mod` + マウスでフローティングウィンドウをドラッグ移動

* **レイアウト / ギャップ**

  * `gaps inner 8`, `gaps outer 0` で、ウィンドウの間だけ 8px の隙間
  * `smart_gaps on` で、ウィンドウ1枚のときは自動でギャップを消す
  * `smart_borders on` で、1枚だけのときは枠を消す
  * タイル分割は `split h`（横）と `split v`（縦）を `$mod+o` / `$mod+Shift+o` で切替

* **ワークスペース**

  * `$ws1`〜`$ws8` を `1`〜`8` に割り当て
  * `workspace_auto_back_and_forth yes` で同じキー押しで前のワークスペースに戻れる

* **ウィンドウの外観（色）**

  * `client.focused` は背景・枠が黒、文字とインジケータが #d8a657
  * 非フォーカス・緊急状態の色も定義済み
  * `client.background #2B2C2B` でデフォルト背景色

* **特定ウィンドウをフロートにする設定**

  * `alsamixer`、`Galculator`、`GParted`、`Lxappearance`、`qt5ct`、`Pavucontrol`、`Simple-scan`、VirtualBox、`copyq` などを自動でフローティング化
  * `copyq` は中央に表示・600x400サイズ・normal border

* **緊急ウィンドウ**

  * `[urgent=latest] focus` で urgent フラグのついたウィンドウに自動フォーカス

### 自動起動しているもの

起動時（i3 start / restart）に以下を自動で立ち上げ：

* 認証系・デーモン

  * `polkit-gnome-authentication-agent-1`（認証ダイアログ）
  * `xss-lock -- betterlockscreen -l`（スクリーンロック連携）

* 見た目・入力

  * `picom -b`（コンポジタ）
  * `setxkbmap jp`（キーボードレイアウトを日本語）
  * `fcitx5 -d`（IME）
  * `feh --bg-fill ...background.png`（壁紙）
  * `polybar`（shades テーマ起動）

* トレイ常駐系

  * `nm-applet`（ネットワーク）
  * `xfce4-power-manager`（電源管理）
  * `pamac-tray`（アップデート通知）
  * `clipit`（クリップボードマネージャ）
  * `copyq --start-server`（もう1つのクリップボードマネージャ）

* Firefox テーマとカーソルテーマの調整

  * `ff-theme-util`
  * `fix_xcursor`

## キーバインド一覧（単体で）

### 起動・ランチャ・アプリ

* `$mod+Return` : wezterm を起動
* `$mod+d` : rofi（combi）起動
* `$mod+z` : `morc_menu` 起動
* `$mod+Ctrl+b` : ターミナルで `bmenu`
* `$mod+F2` : Brave ブラウザ
* `$mod+F3` : Thunar
* `$mod+Shift+F3` : `pkexec thunar`（rootで Thunar）
* `$mod+F5` : ターミナルで `mocp`
* `$mod+p` : VSCode
* `$mod+Shift+n` : ターミナルで `nmtui`
* `$mod+Escape` : wezterm で `btop`
* `$mod+Shift+Escape` : `lxtask`
* `$mod+Shift+x` : Manjaro i3 のヘルプPDFを開く
* `$mod+Ctrl+x` : `xkill`

### サウンド・明るさ・メディア

* `XF86AudioRaiseVolume` : pamixer で音量 +5%、通知
* `XF86AudioLowerVolume` : 音量 -5%、通知
* `XF86AudioMute` : ミュートトグル、通知
* `XF86AudioPlay` : `playerctl play-pause`
* `XF86MonBrightnessUp` : brightnessctl で明るさ +10%、通知
* `XF86MonBrightnessDown` : 明るさ -10%、通知
* `$mod+Ctrl+m` : ターミナルで alsamixer

### スクリーンショット

* `Print` : `screenshot.sh region`（範囲）
* `$mod+Print` : `screenshot.sh full`（全画面）
* `$mod+Shift+Print` : `screenshot.sh window`（アクティブウィンドウ）

### 通知・クリップボード

* `$mod+Shift+d` : dunst 再起動＆「Restarted dunst」通知
* `$mod+v` : copyq のウィンドウ表示

### ウィンドウ操作（フォーカス・移動）

* フォーカス移動（vim 風）

  * `$mod+h/j/k/l` : 左 / 下 / 上 / 右にフォーカス
  * `$mod+Left/Down/Up/Right` : 同じく矢印版

* ウィンドウ移動

  * `$mod+Shift+h/j/k/l` : 左 / 下 / 上 / 右へ移動
  * `$mod+Shift+Left/Down/Up/Right` : 同じく矢印版

* ウィンドウを kill

  * `$mod+Shift+q` : フォーカス中のウィンドウを kill

### フルスクリーン・レイアウト

* `$mod+f` : フルスクリーン切り替え
* `$mod+s` : stacking レイアウト
* `$mod+w` : tabbed レイアウト
* `$mod+e` : split toggle
* `$mod+Shift+space` : 浮動 / タイル切り替え
* `$mod+space` : タイル / フロート間でフォーカス切替
* `$mod+Shift+s` : sticky トグル
* `$mod+a` : 親コンテナにフォーカス

### スプリット方向

* `$mod+o` : `split h`（横方向）＋通知 `'tile horizontally'`
* `$mod+Shift+o` : `split v`（縦方向）＋通知 `'tile vertically'`
* `$mod+q` : split toggle

### スクラッチパッド

* `$mod+Shift+minus` : フォーカスウィンドウをスクラッチパッドへ
* `$mod+minus` : スクラッチパッドのウィンドウを表示 / ローテーション

### ワークスペース

* 移動

  * `$mod+1…8` : ワークスペース 1〜8 に移動
  * `$mod+b` : 前のワークスペースへ戻る

* ウィンドウをワークスペースに送る

  * `$mod+Ctrl+1…8` : フォーカスしているコンテナをだけ移動
  * `$mod+Shift+1…8` : コンテナを移動して、そのワークスペースに移動

* back_and_forth

  * `$mod+Shift+b` : フォーカス中コンテナを前回ワークスペースに送りつつ、そのワークスペースに移動

### 設定リロード・i3 自身

* `$mod+Shift+c` : config のリロード
* `$mod+Shift+r` : i3 を再起動（レイアウト維持）

### システムメニュー（電源・ロック等）

* `$mod+0` : `notify-system-mode.sh` を実行し、`$mode_system` へ入る

モード `$mode_system` 内のキー:

* `l` : 「Lock?」ダイアログ → OK で betterlockscreen
* `s` : 「Suspend?」→ OK で `suspend-lock.sh`
* `u` : 「Switch user?」→ OK で `loginctl lock-session && gdmflexiserver`
* `e` : 「Logout?」→ OK で `i3exit logout`
* `r` : 「Reboot?」→ OK で `systemctl reboot`
* `Shift+s` : 「Shutdown?」→ OK で `systemctl poweroff`
* `Enter / Escape` : モード終了

### リサイズモード

* `$mod+r` : `mode "resize"` に入る

モード `"resize"` 内:

* `h/l/j/k` : 左右上下に 5px / 5ppt ずつサイズ変更
* `Left/Right/Up/Down` : 左右上下に 10px / 10ppt ずつサイズ変更
* `Enter / Escape` : モード終了

### ギャップ調整モード

* `$mod+Shift+g` : ギャップモード `$mode_gaps` へ入る

モード `$mode_gaps`:

* `o` : outer gap 調整モードへ
* `i` : inner gap 調整モードへ
* `Enter / Escape` : デフォルトに戻る

モード `$mode_gaps_inner`:

* `+ / - / 0` : 現在のワークスペースの inner gap を +5 / -5 / 0
* `Shift++ / Shift+- / Shift+0` : 全ワークスペースの inner gap を変更
* `Enter / Escape` : デフォルトへ

モード `$mode_gaps_outer` も同様に outer gap を操作。

### その他

* `$mod+m` : i3bar の表示 / 非表示切り替え
* `$mod+t` : `pkill picom`（コンポジタ停止）
* `$mod+Ctrl+t` : `picom -b`（コンポジタ起動）
